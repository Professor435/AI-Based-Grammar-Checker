## GrammarAI - Features Overview

### Core Features
| Feature | Description |
|---------|-------------|
| **Spelling Check** | Detects 200+ common misspellings (accomodate→accommodate, definately→definitely, etc.) |
| **Grammar Check** | Fixes contractions (dont→do not, cant→cannot, youre→you are) |
| **Punctuation** | Detects double spaces, misplaced punctuation |
| **Style & Tone** | Suggests stronger words (very good→excellent, very bad→terrible) |
| **Wordy Phrases** | Flags filler phrases ("in order to"→"to", "due to the fact that"→"because") |
| **Paraphrasing** | 5 modes: Standard, Fluent, Creative, Formal, Simple |
| **Grammar Score** | Animated circular score with 4 sub-scores (Spelling, Grammar, Style, Clarity) |
| **Real-time Stats** | Word count, character count, reading time, sentences, paragraphs |
| **Issue List** | Clickable issues with Accept/Ignore buttons |
| **History** | Saves last 10 checks to localStorage |
| **Export** | TXT, DOC, PDF export + Copy to clipboard |
| **Undo/Redo** | Full text history management |
| **Keyboard Shortcuts** | Ctrl+Z (Undo), Ctrl+Y (Redo), Ctrl+B/I/U (Format) |

### Design Features
- **Dark theme** with animated gradient orbs background
- **Grid pattern overlay** for tech aesthetic
- **Glassmorphism** toolbar with backdrop blur
- **Animated score circle** with SVG stroke animation
- **Hover effects** on all interactive elements
- **Toast notifications** for user feedback
- **Loading animation** during analysis
- **Responsive layout** (works on mobile & desktop)
- **Tooltips** on toolbar buttons

A fully functional, client-side AI grammar checker built with pure HTML, CSS, and JavaScript. No backend required - works entirely in your browser!

## Features

- **Intelligent Spell Checking** - 200+ commonly misspelled words detected
- **Grammar Correction** - Fixes contractions, subject-verb agreement, tense issues
- **Style Suggestions** - Replaces weak intensifiers with stronger alternatives
- **Punctuation Detection** - Finds double spaces, misplaced commas, etc.
- **Paraphrasing Tool** - 5 rewriting modes (Standard, Fluent, Creative, Formal, Simple)
- **Grammar Score** - Animated scoring with Spelling, Grammar, Style & Clarity breakdowns
- **Real-time Analysis** - Auto-checks as you type with debounced detection
- **Export Options** - Download as TXT, DOC, or PDF
- **Check History** - Automatically saves last 10 checks to localStorage
- **Keyboard Shortcuts** - Ctrl+Z/Y for undo/redo, Ctrl+B/I/U for formatting
- **Responsive Design** - Works on desktop, tablet, and mobile

## How to Use

1. Open `grammar_ai.html` in any modern browser
2. Type or paste your text in the editor
3. Click "Check Grammar" or wait for auto-analysis
4. Review issues in the sidebar and click "Accept" to apply corrections
5. Switch to "Paraphrase" tab to rewrite your text in different styles

## Tech Stack

- **Frontend**: HTML5, CSS3 (CSS Variables, Grid, Flexbox, Animations)
- **Logic**: Vanilla JavaScript (ES6+)
- **Storage**: localStorage for history persistence
- **Design**: Dark theme with glassmorphism, animated gradients, responsive layout

## Browser Support

Chrome, Firefox, Safari, Edge (latest versions)
